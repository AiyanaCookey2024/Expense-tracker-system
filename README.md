# Expense Tracker System

## Overview
This project is a full-stack web application built using Django (backend) and React (frontend). It allows users to manage their budgets and expenses, providing insights into their remaining balance for a given salary period.

The system demonstrates secure authentication, RESTful API design, and full-stack deployment using modern tools.

---

## System Architecture

- **Frontend:** React (Vite)
- **Backend:** Django REST API
- **Database:** PostgreSQL (production), SQLite (development)
- **Deployment:** Render
- **CI/CD:** GitHub Actions

---

## Live Application

- Frontend: https://expense-tracker-system-1-dhpr.onrender.com  
- Backend API: https://expense-tracker-system-1l5v.onrender.com  

---


## Features

- User authentication (JWT)
- Budget and expense management
- Salary tracking
- Password reset functionality
- Cloudinary image uploads
- Fully tested API

---

## Deployment

Both frontend and backend are deployed using Render with environment variables configured securely.
